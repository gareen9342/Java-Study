연산자 : 연산을 하는 것, 연산 부호 등    
피연산자 : 연산 대상

연산자 우선순위가 중요하다. 

**산술 > 논리 > 비교 > 대입**

## 연산자 종류
- 산술 연산자 
- 비교 연산자
- 논리 연산자
- 대입 연산자
- 삼항, instance of, 형변환


## 연산자 우선순위

대부분 상식적으로 알 수 있는 수준인데
아래는 알아두면 좋다

1. 쉬프트 연산자는 덧셈 연산자보다 우선순위가 낮다.  
2. 비트 연산자는 비교 연산자보다 우선순위가 낮다. 
3. 논리 연산자중 && 가 || 보다 우선순위가 높음 

### 연산자 결합규칙
